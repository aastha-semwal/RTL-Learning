# Day 14 – Universal Shift Register

## 📖 Objective

Design and simulate a 4-bit Universal Shift Register using Verilog HDL.

---

## 🛠 Features

- Synchronous Reset
- Hold Operation
- Shift Right
- Shift Left
- Parallel Load

---

## 📂 Files

- universal_shift_register.v
- universal_shift_register_tb.v

---

## 📚 Modes

| Mode | Operation |
|------|-----------|
| 00 | Hold |
| 01 | Shift Right |
| 10 | Shift Left |
| 11 | Parallel Load |
